# SM-808 Practice Exercise Solution

This repository contains a solution in Ruby to the exercise at
URL https://github.com/splicers/sm-808

## Running the example

Assuming you have ruby and bundler installed all ready you can run the example
with

```
bundle exec ruby -I app sm_808_example.rb
```

This will loop over the pattern listed in the requirements until you interrupt
it with CTRL + C.

## Tests

Tests can be run with RSpec using

```
bundle exec rspec
```
